我的第一篇博客

用hexo生成博客（win10）

1.下载安装node.js

2.用node -v 及 npm -v 验证是否安装成功

3.使用npm下载cnpm镜像源
具体cmd输入内容如下：
npm install -g cnpm –registry=http://registy.npm.taobao.org

4.用cnpm -v 验证是否成功

5.安装hexo
具体cmd输入内容如下：
cnpm install -g hexo-cli
注意要先安装git

6.建立一个文件夹blog存放博客内容并进入该博客
mkdir E:\blog
cd blog

7.初始化博客
hexo init
利用dir可看见生成的内容

8.启动博客
hexo server

9.新建新的文章
hexo new “文章的名字”

10.用vs code更改source/_post内文章”文章的名字”的内容

11.重新生成修改后的博客
hexo clean
hexo generate
hexo server

将博客部署至github

1.新建一个repository，名字必须为“用户名”.github.io

2.blog目录下装一个git插件
cnpm install –save hexo-deployer-git

3.修改目录下的_config.yml文件的参数
修改该文件最后的#Deployment部分
deploy:
type: git
repo: https://github.com/“用户名”/“用户名”.github.io.git
branch: master

4.部署至远端
hexo deploy
这里要注意git用户名及电子邮件的配置
git config –global user.name “用户名”
git config –global user.email “电子邮件”

5.访问博客
“用户名”.github.io
